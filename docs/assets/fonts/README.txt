FONT FILES — IBM Plex Sans + IBM Plex Mono
==========================================

These 8 woff2 files need to be placed here for zero-tracking local font hosting.

DOWNLOAD (one-time, free, open source SIL OFL licence):
  https://github.com/IBM/plex/releases/latest

From the release, download:
  IBMPlex-Sans.zip  →  extract these into this folder:
    IBM-Plex-Sans/fonts/complete/woff2/IBMPlexSans-Light.woff2
    IBM-Plex-Sans/fonts/complete/woff2/IBMPlexSans-Regular.woff2
    IBM-Plex-Sans/fonts/complete/woff2/IBMPlexSans-Italic.woff2
    IBM-Plex-Sans/fonts/complete/woff2/IBMPlexSans-Medium.woff2
    IBM-Plex-Sans/fonts/complete/woff2/IBMPlexSans-SemiBold.woff2

  IBMPlex-Mono.zip  →  extract these into this folder:
    IBM-Plex-Mono/fonts/complete/woff2/IBMPlexMono-Regular.woff2
    IBM-Plex-Mono/fonts/complete/woff2/IBMPlexMono-Medium.woff2
    IBM-Plex-Mono/fonts/complete/woff2/IBMPlexMono-SemiBold.woff2

Once all 8 files are here, run  hugo server  — fonts load locally, no Google.

ALSO NEEDED for the logo mark:
  Atkinson Hyperlegible Bold (weight 700)
  https://fonts.google.com/specimen/Atkinson+Hyperlegible
  Download the zip, extract:  AtkinsonHyperlegible-Bold.woff2
  Place it in this folder.
