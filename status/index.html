    История на актуализациите¶

    20.07.2025¶

    Обновяване на Ejabberd до версия 25.07¶

    • Филтър за спам с поддръжка на блокирани списъци;
    • Поддръжка за XEP-0485: PubSub Server Information;
    • Поддръжка за по-стари Matrix стаи в ejabberd XMPP <-> Matrix Gateway;

    17.04.2025¶

    Обновяване на Ejabberd до версия 25.04¶

    • Важна корекция на сигурността и няколко корекции на грешки;
    • Корекция в обработката на множество occupant-id;
    • Нова API команда.

    06.03.2025¶

    Обновяване на Ejabberd до версия 25.03¶

    • Matrix Gateway вече поддържа стаи;
    • Едновременна поддръжка на няколко типа пароли;
    • Възможност за изпълнение на API команди през XMPP клиент.

    26.12.2024¶

    Обновяване на Ejabberd до версия 24.12¶

    • Подобрения и корекции на грешки;
    • XEP-0484: Fast Authentication Streamlining Tokens - Намалява времето за удостоверяване, като осигурява по-бърз старт за клиенти на мобилни устройства;
    • Commands API v3.

    27.11.2024¶

    Ново лого¶

    30.10.2024¶

    Обновяване на Ejabberd до версия 24.10¶

    • XEP-0288: Двупосочни s2s връзки;
    • XEP-0480: Задачи за надграждане на SASL;
    • IQ права в привилегировани обекти;
    • Различни поправки на PubSub;
    • Подобрена уеб базирана администрация.

    22.07.2024¶

    Обновяване на Ejabberd до версия 24.07¶

    • Предимно корекция на грешки възникнали с последната актуализация към версия 24.06;
    • Няколко подобрения.

    29.06.2024¶

    Обновяване на Ejabberd до версия 24.06¶

    • Подобрена уеб базирана администрация;
    • Подобрени архитектура и API.

    29.02.2024¶

    Обновяване на Ejabberd до версия 24.02¶

    • Ново: Matrix Federation - Предоставя възможност за свързване с Matrix сървъри;
    • Подобрена сигурност на комуникацията чрез поддръжка на TLS 1.3 и SASL2;
    • Подобрена производителност, чрез внедряване на Bind 2;
    • По-добър контрол върху съобщения: Поддръжка на XEP-0424 (оттегляне на съобщения). Позволява на потребителите да управляват хронологията на своите съобщения и да премахват нещо, което са публикували по погрешка;
    • Оптимизиран сървърен "ping" чрез XEP-0198; – Рационализиране управлението на API версиите;
    • Подобрена поддръжка на Elixir, Mix и Rebar3.

    14.11.2023¶

    Обновяване на Ejabberd до версия 23.10¶

    • Добавена е поддръжка за XEP-0402: PEP Native Bookmarks, което е значително подобрение спрямо XEP-0048: Bookmark Storage. Определя начина, по който се съхраняват отметките;
    • Ново: mod_muc_occupantid с поддръжка за XEP-0421: Occupant Id. Използва се за анонимни стаи MUC, коригиране на съобщения и оттегляне на съобщения. В момента единственият клиент, за който е установено, че поддържа XEP-0421, е Dino - версия 0.4 и по-нови;
    • Много нови опции и функции. За повече информация, вижте официалната статия в сайта на ProcessOne за новостите във версия 23.10.

    26.04.2023¶

    Обновяване на Ejabberd до версия 23.04¶

    • Много подобрения в SQL базата данни;
    • Ново: mod_mam поддръжка на XEP-0425: Message Moderation;
    • Ново: mod_muc_rtbl поддръжка на Real-Time Block List for MUC rooms.

    20.01.2023¶

    Обновяване на Ejabberd до версия 23.01¶

    • Ново: Отворена е регистрацията, посредством онлайн формата или инсталиран клиент, защитена чрез CAPTCHA предизвикателство.

    Обновяване на Ejabberd до версия 22.10¶
